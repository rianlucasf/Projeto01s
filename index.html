Ciclo de Vida de um Arquivo no Git

1. Untracked (Não Rastreado)

O arquivo ainda não está sendo monitorado pelo Git. É novo no projeto.

2. Modified (Modificado)

O arquivo já foi commitado antes, mas sofreu alterações após isso.

3. Staged (Preparado)

O arquivo foi adicionado com git add e está pronto para ser commitado.

4. Committed (Commitado)

As mudanças foram salvas no histórico do repositório com git commit.

Fluxo completo:
Untracked → Modified → Staged → Committed
